How to Use the Slideshow

The Zaad Academy Event Slideshow module is designed to easily browse through event highlights. Below are the instructions on how to use it.

1. Changing Slides:
Slides will transition automatically. To navigate manually, use the arrows on the screen.

2. Pausing the Slideshow:
Hovering your mouse cursor over any image will pause the automatic rotation.

3. Full Screen Mode:
To view an image in full screen, simply double-click on the image.
